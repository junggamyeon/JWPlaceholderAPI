# JWPlaceholderAPI (Python)

Unified placeholder framework for Endstone plugins.
This is a Python rewrite of the original C++ version of PlaceholderAPI for Endstone.

## Features
- `/papi list`
- `/papi info <id>`
- `/papi parse <player> <text...>`
- `/papi ecloud <list|download <name>|refresh>`
- Custom `ecloud` support using python scripts instead of shared libraries.
- API for other python plugins to register expansions!
